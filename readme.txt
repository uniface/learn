This environment is for use with the Uniface eLearning material on https://learn.uniface.com.

Some starting development objects have been pre-imported from export/uniface 10 learning.xml, 
and data has been imported into a single-user SQLite database from TestData\*.xml.

To compile the provided development objects, use the "IDE - Uniface 10 Learning" shortcut to start the IDE, and at the prompt, type /all and press enter.

Further information on installation and configuration can be found on the web site.
